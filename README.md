# Trivia App
A small trivia application that retrieves questions from the 
[OpenTDB](https://opentdb.com/) API and presents them to the user.

### Setup

This project is built using:
* Java Spring Boot backend
* Vue frontend
* Docker + Docker Compose deployment

### Run the application
1. Clone or download repo
2. Deploy and run the application with `docker compose up --build`
3. Go to [localhost:5173](http://localhost:5173)

To inspect the backend, go to [http://localhost:8080/docs](http://localhost:8080/docs).
